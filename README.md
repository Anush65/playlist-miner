# 🎧 PlaylistMiner

Full-stack tool to search, preview, and download YouTube music as MP3/MP4.

- 🔍 Search songs from YouTube
- 🎥 Live preview of top result
- 📥 Download audio/video
- 🌐 Built with FastAPI + React + Tailwind
